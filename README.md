# Лендинг Алексея Рыкова

Тёплый одностраничный сайт-портфолио для реставрации и оживления семейных фотографий.

## Запуск

Откройте `index.html` в браузере или запустите любой статический сервер в папке проекта.

## Что заменить перед публикацией

1. Укажите реальные ссылки Telegram, WhatsApp, MAX и VK в блоке `social-buttons` файла `index.html`.
2. Замените демонстрационное изображение `assets/grandmother-restored.png` реальными работами.
3. Замените демонстрационные отзывы на настоящие.
4. При необходимости уточните цены и сроки.

Сравнение «до/после» работает интерактивно. Левая версия создаётся из того же снимка стилями, поэтому для настоящего кейса лучше добавить два отдельных файла и заменить соответствующие фоновые изображения.
